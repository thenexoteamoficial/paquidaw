# 🎮 Paquidaw

**Videojuego 3D inspirado en el universo de Halo**  
Desarrollado **100% en Python** usando el motor de juegos **Ursina**.

![Python](https://img.shields.io/badge/Python-3.10+-blue?style=flat-square)
![Ursina](https://img.shields.io/badge/Ursina-Engine-orange?style=flat-square)

## ✨ Características principales

- Gráficos 3D de alta calidad
- Experiencia de juego tipo FPS inspirada en Halo
- Soporte de texto a voz (pyttsx3)
- Optimizado para Windows
- Próximamente: Android y más plataformas

## 🛠️ Tecnologías utilizadas

- **Python 3**
- **Ursina** - Motor de juegos 3D ligero y potente
- **pyttsx3** - Síntesis de voz

## 📦 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/thenexoteamoficial/paquidaw.git
cd paquidaw

# Instalar dependencias
pip install -r requirements.txt
```

## ▶️ Cómo ejecutar el juego

```bash
python server.py
```

> **Nota**: El archivo principal es `server.py` (próximamente se renombrará a `main.py` para mayor claridad).

## 📦 Crear ejecutable (.exe) para Windows

```bash
pip install pyinstaller
pyinstaller --onefile --windowed server.py
```

El ejecutable aparecerá en la carpeta `dist/`.
