# 🎛️ PaquiDAW

DAW (Digital Audio Workstation) hecho con Python + HTML

## 🚀 Funciones
- 🎧 Multitrack
- 🎤 Grabación
- 🎹 Instrumentos virtuales
- 🧠 IA (base para Demucs)

## ▶️ Cómo usar
1. Instalar dependencias:
pip install -r requirements.txt

2. Ejecutar servidor:
python server.py

3. Abrir index.html
